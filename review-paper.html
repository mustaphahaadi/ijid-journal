Transformer Architecture for Code Generation Tasks

Paper ID: #IJID-2026-0078

Assigned: Feb 25, 2026

Deadline: Mar 15, 2026

Authors: S. Chen, W. Liu, T. Park

Institution: Stanford University

Category: Artificial Intelligence

Type: Original Research

Abstract:

This paper presents a novel transformer-based architecture optimized for automated code generation tasks across multiple programming languages. We introduce CodeFormer, a model that leverages hierarchical attention mechanisms and syntax-aware positional encodings to generate syntactically correct and semantically meaningful code from natural language descriptions. Our experiments on the HumanEval benchmark demonstrate a 15.3% improvement over existing state-of-the-art methods, with particularly strong performance on complex multi-step programming tasks. We also introduce a new evaluation metric, Semantic Correctness Score (SCS), which better captures the functional equivalence of generated code.

Keywords: Transformer Code Generation NLP Deep Learning
